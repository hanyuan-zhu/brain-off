# 🤖 AI 任务管理助手 - 快速开始

## 启动聊天界面

```bash
python chat.py
```

## 可用命令

- `/help` - 显示帮助信息
- `/clear` - 清除对话历史
- `/stats` - 显示会话统计
- `/exit` - 退出聊天

## 示例对话

### 1. 捕获想法
```
你: 我想做一个个人财务管理工具，能自动分类支出
🤖 助手: ✓ 已捕获你的想法！
```

### 2. 搜索想法
```
你: 找一下关于财务的想法
🤖 助手: 找到 3 个相关想法：
1. 个人财务管理工具...
2. 家庭预算追踪...
```

### 3. 建议标签
```
你: 帮我的财务工具想法建议一些标签
🤖 助手: 建议标签：财务管理、个人工具、自动化
```

## 功能特性

✅ **想法捕获** - 随时记录你的灵感
✅ **语义搜索** - 用自然语言搜索历史想法
✅ **AI 标签建议** - 自动建议合适的标签
✅ **对话式交互** - 像聊天一样管理任务

## 技术亮点

- 🧠 **向量搜索** - PostgreSQL + pgvector 实现语义搜索
- 🤖 **AI 驱动** - DeepSeek API + DashScope embeddings
- 🔄 **Agent 循环** - Query → Thoughts → Actions → Review
